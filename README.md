# Real Estate Management System

A comprehensive web-based system designed to manage real estate operations, including properties, units, tenants, contracts, and financial transactions.

## Key Features
- Property and unit management
- Rental contract management with payment scheduling
- Automated rent tracking and overdue alerts
- Revenue and expense reporting (office & property level)
- Tenant management and contract history
- Financial dashboards and insights

## Technologies
- ASP.NET Web Forms
- C#
- SQL Server
- RDLC Reports
- JavaScript, Bootstrap 5
- HTML, CSS

## My Contribution
- Designed and developed the full system architecture
- Implemented rental tracking and alert mechanisms
- Built dynamic financial reports using RDLC
- Optimized database structure for performance and scalability

## Availability
Private repository — Developed for a private company - Source code is not publicly available.

# نظام إدارة العقارات

نظام ويب متكامل لإدارة العمليات العقارية، يشمل إدارة العقارات والوحدات والمستأجرين والعقود والمعاملات المالية.

## المميزات الرئيسية
- إدارة العقارات والوحدات
- إدارة عقود الإيجار وجدولة الدفعات
- متابعة الإيجارات والتنبيه عند التأخير
- تقارير الإيرادات والمصروفات (للمكتب والعقار)
- إدارة بيانات المستأجرين وسجل العقود
- لوحات تحكم مالية وتحليلات

## التقنيات المستخدمة
- ASP.NET Web Forms
- C#
- SQL Server
- RDLC Reports
- JavaScript, Bootstrap 5
- HTML, CSS
